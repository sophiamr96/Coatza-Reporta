# COATZA REPORTA

Coatza Reporta es una aplicación de escritorio desarrollada en C# y Windows Forms (.NET Framework 4.8) que permite registrar, administrar y dar seguimiento a reportes ciudadanos relacionados con problemáticas urbanas en la ciudad de Coatzacoalcos
El sistema facilita la captura de incidencias como:
- Alumbrado
- Baches
- Basura
- Contaminación
- Drenaje
- Fugas
- Obra pública
- Obstrucciones
- Ruido
- Seguridad
- Semáforos

Cada reporte incluye información del ciudadano, ubicación, referencias y evidencia fotográfica.

## Objetivo

Proporcionar una herramienta que permita facilitar el registro, organización y seguimiento de reportes ciudadanos en la ciudad de Coatzacoalcos.

## Tecnologías Utilizadas
- C#
- .NET Framework 4.8
- Windows Forms
- ClosedXML
- PDFSharp
- Microsoft Edge WebView2
- Excel (.xlsx)

## Funcionalidades

##### Registro de reportes
- Captura de datos del ciudadano
- Selección del tipo de reporte
- Dirección estructurada por colonia y calles
- Referencias adicionales
- Carga de evidencia fotográfica

##### Generación de PDF
- Creación automática de reportes en PDF
- Folio único para identificación

##### Almacenamiento
- Guardado automático en archivo Excel
- Generación automática de carpetas necesarias

##### Panel administrativo
- Visualización de reportes mediante DataGridView
- Filtrado por tipo, fecha y estado del reporte

##### Seguimiento
- Actualización de estado del reporte (Pendiente, En curso, Resuelto)

##### Dashboard
- Reportes pendientes
- Reportes en curso
- Total de Reportes
- Reportes registrados hoy
- Estadísticas por tipo de reporte (Gráfica dinámica)

- Instanciación de clases

- Interfaces
